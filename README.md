# 🎮 Shape Clicker Game 🎮

Welcome to the **Shape Clicker Game**! This is a fun, simple clicker game built with HTML, CSS, and JavaScript. Test your reflexes by clicking on randomly appearing shapes before they disappear!

## 🎯 How to Play
- 🔴 **Circles**, 🔵 **Squares**, and 🔺 **Triangles** will appear randomly in the game area.
- 🖱️ **Click** on them as fast as possible to score points!
- ⏱️ Each shape disappears after 5 seconds if not clicked.
- 📊 Your score increases with every successful click.
- ⚠️ **Game Over** if you miss 3 shapes – watch out! 💀

## 🚀 Features
- 🎨 Colorful shapes: Red circles, blue squares, green triangles.
- 🕒 Timed gameplay with automatic shape spawning.
- 📱 Responsive design (works on different screen sizes).
- 🎉 Real-time score updates.

## 🛠️ Technologies Used
- 🌐 **HTML** for structure
- 🎨 **CSS** for styling
- ⚙️ **JavaScript** for game logic

## 📁 Files
- `index.html` - Main game page
- `style.css` - Game styling
- `script.js` - Game functionality
- `README.md` - This file! 📖

## 🎲 Getting Started
1. Open `index.html` in your web browser.
2. Start clicking shapes and enjoy! 😄
